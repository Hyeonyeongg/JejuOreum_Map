# 오름 지도

제주도의 오름 위치 및 각각의 오름에 대한 간단한 정보를 얻을 수 있는 웹 서비스

- html, css, JavaScript

<br/>

## <sapn id="intro"> 3. 핵심 포인트 </span>

- 오픈 API를 통해 제주에 있는 90개의 오름에 대한 정보 활용
- Google Maps API를 통해 마커로 오름의 위치를 표시하고, 오름의 높이를 파란색 원으로 표현
- 오름에 대한 정보를 모달창으로 표현
- API 사용에 익숙해지기 위한 목적으로 프로젝트를 제작함.
